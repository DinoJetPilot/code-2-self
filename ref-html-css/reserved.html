code explanation
< opening HTML tag
> closing HTML tag
& used for entity codes
" " used for the value of a certain attribute

An entity is interpreted by HTML to display a certain character or symbol. It begins with & and ends with ; (with either caracters or numbers in the middle).

The <code> HTML element is intended for inline code. Usually parts of code without needing line breaks.

The <pre> HTML element is displayed as monospaced, preformatted text and retains line specing and breaks.
